📈 Trading Bot Simulator
This is a Python-based simulation tool designed to analyze stock market trends. It uses technical indicators to provide a visual representation of market movements.

🚀 Key Features
Technical Indicators: Implements RSI (Relative Strength Index) and MACD (Moving Average Convergence Divergence) logic.

Data Simulation: Processes historical data to simulate buy/sell signals.

Analysis: Helps in understanding market momentum and trend reversals.

🛠️ Built With
Python 3.x

Pandas: For data manipulation.

Matplotlib: For charting and visualization.
